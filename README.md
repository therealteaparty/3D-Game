# 3D-Game
Project 5 for MSCH-C220

#Implementation
Original Flashlight made with Blender (Enter F to toggle ON/OFF)

Additional Features:
	1. Comprehensive HUD
	2. Particles
	3. Collectable objects
	4. Original handheld object model, enemy model
	5. OST
